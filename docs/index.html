Generalidades

La aplicación consiste en una tienda online que despliega productos agrupados por categorías y que cuenta con un buscador para desplegar productos más específicos.

Get Started

git clone https://github.com/DanyBeth-Dev/backend-bsale.git

Para usar localmente, clone el repositorio en su equipo y en su terminal ejecute dentro de la ubicación del archivo los comandos `npm install` para descargar los módulos necesarios.

Arquitectura

Se encuentra desarrollada bajo arquitectura MVC.

  • Modelo: Consiste en la Base de Datos MySQL alojada en la nube de AWS.
  • Vista: Consiste en el Frontend el cual se encuentra alojado en GitHub Pages.
  • Controlador: Consiste en Backend alojado en Heroku en un dyno gratuito.

Frontend

Desarrollado en Vanilla JavaScript, utilizando manipulación del DOM con técnicas directas y a través de templates y fragments.

Url:

  • https://www.daniela-dev.cl/frontend-bsale

Backend

Desarrollado en Node.js y Express, en conjunto con MySQL.

Url:

  • https://evening-taiga-78339.herokuapp.com

Endpoints:

  • GET '/data/'

Entrega todos los productos de la tabla product utilizando la query `'SELECT * FROM product'`

  • GET '/data/:producto'

Entrega rutas personalizadas para cada producto en base a la categoría a la que pertenecen, utilizando la query: `SELECT * from product WHERE category IN (SELECT id FROM category WHERE name = "${producto}")`

  • GET '/search'

Realiza una búsqueda utilizando query strings en base al parámetro ingresado en el input search del lado del cliente. Su query es: `SELECT * FROM product WHERE name LIKE "%${producto}%"`
